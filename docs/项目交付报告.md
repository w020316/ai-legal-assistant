## 一、项目概述

AI 法律助手是一款面向普通用户与法律从业者的智能法律咨询平台，提供 AI 智能问答、案例检索、法律文书模板生成、合同文档风险分析、个人知识库管理等核心功能。

### 技术架构

| 层级 | 技术栈 |
|------|--------|
| 前端 | Vue 3 + TypeScript + Pinia + Element Plus + Vite |
| 后端 | Spring Boot 3.4 + Spring AI 1.0.0 GA + MyBatis-Plus |
| 数据库 | PostgreSQL + pgvector（向量检索） |
| 缓存 | Redis（JWT 黑名单 / 会话缓存） |
| AI | OpenAI V1 兼容 API（AgnesClient） |
| 部署 | Fly.io（前端 + 后端 + PostgreSQL + Redis） |

### 访问地址

| 服务 | 地址 |
|------|------|
| 前端 | https://lawai-frontend.fly.dev/ |
| 后端 API | http://lawai-backend.fly.dev/api/v1/health |
| 登录凭据 | admin / admin123 |
